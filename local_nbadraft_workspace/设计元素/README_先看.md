# 个人设计资产库

这个文件夹是一套可复用的设计 skill 资产库,不按项目来源组织,只按“设计判断 / 视觉元素 / 动效元素 / 界面组件 / 验收规则”组织。

## 先读顺序

1. `资产总览.md`
2. `01_设计规范/`
3. `03_动效资产/`
4. 按任务类型读取 `02_视觉元素/`、`04_界面组件/`、`05_验收规则/`

## 使用原则

- 不问“它原来属于哪个项目”,只问“现在这个任务需要什么设计效果”。
- 每个元素都写清楚适用场景、使用方式、禁忌和验收点。
- 做 skill 时,`SKILL.md` 只放触发和流程,这些文件放进 `references/` 供按需读取。

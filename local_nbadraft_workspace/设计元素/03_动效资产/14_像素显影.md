# 像素显影

## 适合

切歌封面、图片切换、作品图切换、媒体卡片更新。

## 效果

图片切换时短暂模糊、饱和、对比、微放大,再快速恢复清晰,像从像素/缓存中显影出来。

## 实现抓手

- 状态变化时设置 decay 值。
- rAF 600ms ease-out 衰减。
- `filter: blur(...) saturate(...) contrast(...)`。
- `transform: scale(...)`。
- 可升级为 SVG filter / feMorphology / pixel grid。

## 禁忌

- 不要用于正文或 UI 图标。
- 不要时间过长。
- 不要过度 blur 导致切换时像加载失败。

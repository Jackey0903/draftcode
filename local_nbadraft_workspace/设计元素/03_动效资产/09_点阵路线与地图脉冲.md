# 点阵路线与地图脉冲

## 适合

地图、导航、物流路径、数据流、驾驶状态、系统活跃状态。

## 效果

点阵地图底纹 + 路线点流动 + 当前点 halo 脉冲 + 状态改变时速度/亮度变化。

## 实现抓手

- 背景用 `radial-gradient` 生成点阵。
- route dots 用 absolute 小圆点沿路径排布。
- 每个点做 `opacity + scale` 循环。
- marker halo 用 `scale + opacity` 向外扩散。
- 用 class 表示 calm/focused/tired/drive/auto 等状态。

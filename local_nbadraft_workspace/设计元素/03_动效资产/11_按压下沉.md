# 按压下沉

## 适合

所有按钮、chip、卡片按钮、Dock 图标、拨号盘等可点控件。

## 效果

点击瞬间控件轻微下沉或缩小,亮度变暗,模拟物理按压。

## 实现抓手

```css
button:active,
[role="button"]:active {
  transform: translateY(1px);
  filter: brightness(.92);
  transition: transform 80ms ease, filter 80ms ease;
}
```

## 搭配

和 `触摸涟漪` 搭配最好: 一个给局部触感,一个给屏幕反馈。

## 禁忌

- 不要位移过大。
- 不要影响布局尺寸。
- 不要覆盖已有 transform 时忘记合并。

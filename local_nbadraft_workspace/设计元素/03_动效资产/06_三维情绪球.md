# 三维情绪球

## 适合

AI 助手、车机伙伴、智能代理、品牌吉祥物、情绪化产品对象展示。

## 效果

一个哑光球体带边缘 fresnel、细噪点、双眼光点、呼吸、跟随鼠标、偶尔眨眼,旁边可配能力卡。

## 实现抓手

- Three.js `SphereGeometry`。
- 自定义 shader: base 色、顶部柔光、fresnel 边缘光、grain。
- 双眼用小球体 + additive halo。
- 外环/底座用 torus/cylinder + additive blending。
- 鼠标跟随用 lerp。
- 呼吸用 `sin(t)` 轻微 scale。
- 视口外停止 render。

## 禁忌

- 不要当普通装饰球。
- 不要同时放太多 3D 对象。
- 不要牺牲性能追求细分数。

## 降级

静态 PNG / CSS 径向球体 / 简化 SVG 光环。

# 全屏擦除 Wipe

## 适合

模式切换、场景切换、系统状态接管、从普通界面进入沉浸界面。

## 效果

一层半透明主题色膜从左到右/从下到上擦过,中途替换场景,擦除后消失。

## 实现抓手

- 固定层 `#wipe-layer`。
- 初始 `clip-path: polygon(0 0, 0 0, 0 100%, 0 100%)`。
- wiping: 展开到全屏。
- 中途切换状态。
- fading: 收到另一侧并 opacity 归零。

## 推荐参数

- 展开 500ms。
- 状态切换在 50% 或展开结束时发生。
- `cubic-bezier(.7,0,.3,1)`。

## 禁忌

- 不要每次小按钮点击都 wipe。
- 不要用纯不透明层造成闪断。

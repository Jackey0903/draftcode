# FusionPixel12

## 文件

- `fusion-pixel-12px-monospaced-zh_hans.otf.woff2`
- `OFL.txt`

## 用法

这就是 `QUANTDRAFT 30 · 选秀作战室` 封面竖排像素风“选秀作战室”使用的字体资产。

建议在 CSS 里声明为:

```css
@font-face{
  font-family:"FusionPixel12";
  font-style:normal;
  font-weight:400;
  font-display:swap;
  src:url("字体资产/FusionPixel12/fusion-pixel-12px-monospaced-zh_hans.otf.woff2") format("woff2");
}
```

## 使用纪律

- 只用于状态、编号、日志、短竖排字标、故障/数据雨/系统启动感场景。
- 不用于正文、长句、说明段落、表格说明。
- 页面没有明确要求像素风时,统一使用得意黑。

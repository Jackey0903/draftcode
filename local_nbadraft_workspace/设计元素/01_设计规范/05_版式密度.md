# 版式密度

## 核心规则

一屏只选一个主版式。内容不适配版式,就拆屏,不要压缩文字。

## 可选主版式

- Cover
- Section Header
- Text + List
- Metrics / Grid
- Single Visual
- Horizontal Split
- Timeline
- Comparison
- Quote
- Diagram / Teardown
- Risk / Decision Map

## 密度预算

普通屏幕最多包含:

- 一个焦点元素。
- 一个主版式容器。
- 一到两个辅助块。
- 一个 caption/source/detail 区。

## 常见失败

- 大框里只有几个小节点。
- 三四张文字卡塞进半宽栏。
- 标题、双栏、指标行、脚注挤在同一屏。
- 上半屏很挤,下半屏全空。
- 为了放下内容把字号压到读不清。

## 修正方式

拆屏、缩小框、扩大内容结构、改成全宽 evidence band、减少卡片数量。

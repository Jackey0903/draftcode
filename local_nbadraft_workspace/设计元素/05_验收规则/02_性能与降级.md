# 性能与降级

## 目标

全屏展示或 1920x1080 原型应保持流畅,不要为了高级效果牺牲演示稳定性。

## 高风险效果

- WebGL 噪点。
- Three.js 物体。
- 粒子文字。
- 大量 blur/backdrop-filter。
- 大量 box-shadow/glow。

## 降级规则

- WebGL 背景 -> 静态渐变 + SVG 噪点。
- Three.js 物体 -> PNG 或 CSS 球体。
- 粒子文字 -> 静态大字。
- 视频 -> 720p 版本。
- 强玻璃 -> 降低 blur 或关闭 glass。

## 实现注意

- DPR 限制到 2。
- 视口外停止 render 或跳过绘制。
- 视频 muted + playsinline + preload metadata。
- 动效优先使用 transform、opacity、filter、clip-path。

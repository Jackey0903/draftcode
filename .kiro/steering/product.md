# Product steering

DraftCode is an NBA draft prediction agent for a 24-hour AWS hackathon.

The product must generate final first-round team-player predictions from data and preserve an auditable trace. The user-facing demo should make the prediction logic clear quickly: final picks, confidence, top alternatives, and why a team-player match was selected.

Manual intuition can guide data and weights, but the submitted answer must be produced by the agent pipeline.
